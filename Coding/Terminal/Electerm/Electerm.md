# Electerm

## 概览

要创建同步, 那么有两个步骤需要做:

1. 创建一个Gist ID, 其实就是一个代码片段的仓库

   - [Create secret gist in Github](https://github.com/electerm/electerm/wiki/Create-secret-gist)
   
2. 创建一个Gist 访问的 Token

   - 注意创建出来的Token是全局可用的
