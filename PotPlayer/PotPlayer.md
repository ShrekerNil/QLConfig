# PotPlayer

## 常用快捷键

| 用途                 | 快捷键             | 备注      |
| -------------------- | ------------------ | --------- |
| 查看视频信息         | Tab                |           |
| 上一帧               | D                  |           |
| 下一帧               | F                  |           |
| 关键帧               | Ctrl + Shift + ←/→ |           |
| 减速播放             | X                  | 最低0.2倍 |
| 加速播放/倍速播放    | C                  | 最高12倍  |
| 正常速度播放         | Z                  |           |
| 区间循环播放标记     | []                 |           |
| 取消循环播放标记     | {}                 |           |
| 添加书签             | P                  |           |
| 打开书签界面         | H                  |           |
| 复制当前画面到剪切板 | Ctrl + C           |           |
| 截图设置             | K                  |           |
| 录制视频             | Alt + C            |           |

## 常用功能

1. 解码器切换
   - S/W：软解码
     - 兼容性更强
     - 比较考验CPU性能
   - H/W：硬解码
     - 节能高效
     - 兼容性稍弱，不支持切换比例的缩放
2. 进度条显示缩略图
   - 设置 → 播放 → √ 鼠标指向进度条时显示缩略图

## 常见问题总结

### 倍速播放时音色变高

Shift + X 打开音频滤镜即可

### 画面灰白不清晰

视频渲染器选择：内置 Direct3D 11 视频渲染器

### 声音忽大忽小

设置 → 声音 → 规格化/混响 → 规格化：取消勾选：播放时开启规格化

### 偶尔播放卡顿

解码器切换一下

### 不支持最新的h265编码格式

安装：https://github.com/Nevcairiel/LAVFilters/releases
